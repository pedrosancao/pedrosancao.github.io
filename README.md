## Features

- Speed optimized - [95 mobile / 99 desktop on PageSpeed][pagespeed]
- Typing animation
- Full height content "hero"
- Scroll snapping
- Fixed background/parallax
- Sensitive data protection (e-mail) with recaptcha
- Desaturation/saturation filter on focus
- Multilanguage
